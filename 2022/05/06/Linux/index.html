Linux基础命令

文件目录

  1. pwd 显示当前工作目录的绝对路径

  2. ls 列出目录内容

    按键 功能
    -a 全部的文件，连同隐藏的文件也一起列出来
    -l 显示文件的详细信息， ls -l 等价于 ll

  3. cd 切换目录

    按键 功能
    cd [绝对路径] 切换目录
    cd [相对路径] 切换目录
    cd ~ 或 cd 回到用户目录
    cd - 回到上次所在目录
    cd .. 回到当前目录的上一级目录
    cd -P 回到实际物理路径，而非快捷方式路径

  4. mkdir 创建文件夹

    按键 功能
    -p 创建多层目录

  5. rmdir 删除一个空的目录

  6. touch 创建文件

  7. cp 复制文件或目录

    按键 功能
    -r 递归复制整个文件夹
    /cp 直接覆盖同名文件，不询问

  8. rm 删除文件或目录

    按键 功能
    -r 递归删除目录中所有内容
    -f 强制执行删除操作，不询问
    -v 显示指令的详细执行过程

  9. mv 移动文件与目录或重命名

  10. cat 查看文件内容

  1. more 分屏查看文件内容

    按键 功能
    空格键 向下翻一页
    enter 向下翻一行
    q 退出
    ctrl + f 向下滚动一屏
    ctrl + b 返回上一屏
    = 输出当前行的行号
    :f 输出文件名和当前行号

  2. less 分屏查看文件内容

    按键 功能
    空格键 向下翻一页
    [pagedown] 向下翻一页
    [pageup] 向上翻一页
    /字串 向下搜寻[字串]；n查找下一个，N查找上一个
    ?字串 向上搜寻[字串]；n查找下一个，N查找上一个
    q 退出

  3. echo 输出内容到控制台

    按键 功能
    \\ 输出 \
    \n 换行
    \t 制表符，也就是tab键
    -e 启用转义符

  4. head 显示文件头部内容

    -num 显示头部内容的行数，默认显示10行

  5. tail 显示文件尾部内容

    按键 功能
    -num 显示文件尾部的行数，默认显示10行
    -f 显示文件最新追加的内容，监视文件变化

  6. 输出重定向 > 和追加 >>

  7. ln 软连接

> 类似Windows中的快捷方式

  1. history 查看已经执行过的历史命令

时间日期

关于时间日期

按键 功能
-d [时间字符串] 显示指定的时间字符串表示的时间，而非当前时间
-s [日期字符串] 设置系统日期时间
[+日期时间格式] 使用指定格式显示当前时间
date 显示当前日期
date+%Y 显示当前年份
date+%m 显示当前月份
date+%d 显示当前是哪一天
date “+%Y-%m-%d %H:%M:%S” 显示年月日时分秒

  1. date 显示当前时间

  2. date 显示非当前时间

  3. date 设置系统时间

  4. cal 查看日历

用户管理

  1. useradd 添加新用户

    -g 为用户添加分组

  2. passwd 设置用户密码

  3. id 查看用户是否存在

  4. 查看创建了哪些用户

  5. su 切换用户

  6. userdel 删除用户

    -r 删除用户和用户目录

  7. who 查看登录用户信息

  8. sudo 设置普通用户具有root权限

  9. usermod 修改用户

    -g 修改用户的初始登录组，给定的组必须是存在的

用户组管理

  1. groupadd 新增组

  2. groupdel 删除组

  3. groupmod 修改组

    -n 指定工作组的新组名

  4. 查看有哪些组

文件权限

  1. chmode 改变权限

    这里就只用二进制的方法了

    -R 递归操作，对整个文件夹里的所有内容执行

  2. chown 改变所有者

  3. chgrp 改变所属组

搜索查找

  1. find 查找文件或目录

    按键 功能
    -name 按指定的文件名查找模式查找文件
    -user 查找属于指定用户的文件
    -size 按照指定的文件大小查找文件；+n表示大于，-n表示小于，n表示等于

  2. grep 过滤查找以及 | 管道符

    -n 显示匹配行及行号

解压缩

只写 .tar.gz 了

  1. tar 打包

    按键 功能
    -c 产生 .tar打包文件
    -v 显示详细信息
    -f 指定压缩后的文件名
    -z 打包同时压缩
    -x 解压 .tar文件
    -C 解压到指定目录

磁盘分区

  1. du 查看文件和目录占用的磁盘空间

    按键 功能
    -h 以G，mb，kb等格式显示
    -a 查看子目录及文件大小
    -c 显示所有文件和子目录大小后，显示总和
    -s 只显示总和

  2. df 查看磁盘空间使用情况

进程管理

  1. ps 查看当前系统进程状态

    按键 功能
    -e 列出所有进程
    -f 显示进程的完整信息

  2. kill 终止进程

    -9 表示强制终止

  3. pstree 查看进程树

    按键 功能
    -p 显示进程的PID
    -u 显示进程的所属用户

  4. top 实时监控系统进程状态

    按键 功能
    -d 秒数 指定top命令在每个几秒后进行更新
    -i 使top不显示任何闲置或者僵死的进程
    -p 通过指定进程ID来仅仅监控某个进程的状态
    P 以 CUP 使用率排序 默认
    M 以内存的使用率排序
    N 以 PID 排序
    q 退出top

  5. netstat 显示网络状态和端口占用信息

    按键 功能
    -a 显示所有正在监听(listen)和未监听的嵌套字(socket)
    -n 拒绝显示别名，能显示数字的全部转化成数字
    -l 仅列出在监听的服务状态
    -p 表示显示哪个进程在调用

  3. 开放范围端口
